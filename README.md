# SIMULATION-DEMO

## React + Vite

Please, follow the steps below to run the application locally.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/dirtyuniversalunicorn/SIMULATION-DEMO.git
2. Navigate to the directory:
   ```bash
   cd SIMULATION-DEMO
4. Install all dependencies:
   ```bash
   npm install
5. Frontend application **does not have** any environmental variables.
   
6. Run the development server
   ```bash
   npm run dev
